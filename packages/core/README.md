# `@bromaid/core`

Tokenizer, parser, AST, and structured errors for the **bromaid** diagrams-as-code DSL.

## Install

```bash
npm i @bromaid/core
```

## Usage

```ts
import { parse } from '@bromaid/core';

const program = parse('actor user "User"');
```
